PCA / t-SNE / UMAP Settings

×
When to use each method:
PCA — Best for exploring linear relationships. Fast, deterministic. Shows which variables (loadings) drive separation. Use when groups separate along linear axes.
t-SNE — Reveals clusters in complex, non-linear data. Good for visualizing distinct groups. Stochastic: re-runs may differ. Adjust Perplexity (lower = tighter clusters, higher = more global view). Try values between 5 and 50.
UMAP — Similar to t-SNE but better preserves global structure. Faster on large datasets. Adjust Neighbors (local detail) and Min Dist (cluster tightness). Lower values give tighter, more separated clusters.
Tip: There is no single “correct” parameter value. Try different settings and pick the one that gives the clearest group separation — this is standard practice.
Statistics used:
PCA: Eigendecomposition of the covariance matrix. Data is centered. Variance explained (%) shown per component. Loadings show eigenvector coefficients.
t-SNE: Converts pairwise distances to probabilities, then optimizes a 2D layout where similar points stay close. Stochastic optimization.
UMAP: Builds a nearest-neighbor graph and optimizes a 2D layout preserving both local and global structure. Initialized from PCA coordinates.
Loadings (t-SNE/UMAP): Shows Pearson correlations between original features and embedding coordinates. Note: These are less reliable than PCA loadings because t-SNE/UMAP distort distances non-linearly. Arrows often cluster in similar directions and should be interpreted with caution. PCA loadings are generally preferred for identifying driving variables.
Implementation:
All methods implemented in client-side JavaScript (pca.js). Statistical tests in other modes use jStat.
Methods text for manuscripts:
“Dimensionality reduction was performed using [PCA / t-SNE (perplexity = X) / UMAP (n_neighbors = X, min_dist = X)]. Data were z-score standardized prior to embedding. Visualizations were generated using Visualize (client-side JavaScript).”

Comparing two groups

Unpaired t-test (Welch's)
Compares means of two independent groups. Assumes normality but not equal variances. Most common choice for two-group comparison with reasonably normal data.
Paired t-test
Compares means of matched/repeated measurements (e.g., before vs after). Requires equal sample sizes. Use when each observation in one group has a corresponding observation in the other.
Mann-Whitney U
Non-parametric alternative to unpaired t-test. Compares rank distributions. Use when data is not normally distributed, ordinal, or has outliers.
Wilcoxon signed-rank
Non-parametric alternative to paired t-test. Compares ranks of paired differences. Use for paired data that is not normally distributed.

Comparing three or more groups

One-way ANOVA
Tests whether means differ across 3+ groups. Assumes normality and homogeneity of variances. Follow with post-hoc tests (Tukey, Bonferroni, Holm, or Dunnett) to identify which pairs differ.
Kruskal-Wallis
Non-parametric alternative to one-way ANOVA. Compares rank distributions. Follow with Dunn's test for pairwise comparisons.
Friedman
Non-parametric test for 3+ paired/repeated groups. Rank-based alternative to repeated-measures ANOVA. Requires equal sample sizes.

Time Series

Two-way RM ANOVA (Group × Time)
The standard test for time series data with groups. Tests three effects: Group (do groups differ overall?), Time (does the measurement change over time?), and Interaction (do groups change differently over time?). Post-hoc comparisons use per-timepoint ANOVA as a gatekeeper, followed by pairwise t-tests with multiple comparison correction.

Correlation

Pearson correlation (r)
Measures strength of linear association. Assumes both variables are normally distributed. Values range from −1 (perfect negative) to +1 (perfect positive). Use when data appears linearly related.
Spearman rank correlation (ρ)
Non-parametric measure of monotonic association. No normality assumption. Use when data is ordinal, has outliers, or the relationship is monotonic but not necessarily linear. Shown in Extended stats mode.

Kaplan-Meier Survival Analysis

Kaplan-Meier curves estimate survival probability over time. When you have two or more groups, the log-rank test is automatically computed to test whether the survival curves differ significantly.

Log-rank test
Compares survival distributions between groups. Non-parametric; makes no assumption about the shape of the survival curve. The test statistic (χ²) and p-value are shown on the graph when "Log-rank test" is checked in settings. This is the only test available for Kaplan-Meier data — it is the standard method used in clinical research.

Clustering & outlier detection

Hierarchical clustering
Groups rows and/or columns by similarity using agglomerative clustering. Linkage methods: Average (UPGMA), Complete (furthest neighbor), Single (nearest neighbor). Euclidean distance is used.
Normalization
Z-score (mean/SD) or robust (median/MAD) normalization can be applied per row, per column, or across the whole dataset to make values comparable across different scales.
Winsorization
Clips extreme values to reduce the impact of outliers on the color scale, improving visual contrast for the majority of data.

Post-hoc corrections

Tukey HSD
Standard choice for all pairwise comparisons after ANOVA. Controls family-wise error rate. Balanced in power vs. strictness.
Bonferroni
Most conservative. Divides α by the number of comparisons. Use when strict Type I error control is critical.
Holm-Bonferroni
Step-down method. Less conservative than Bonferroni, more powerful. Generally preferred over Bonferroni.
Dunnett
Compares each group to a single control group only. More powerful than all-pairwise methods when only control comparisons are needed.

Post-hoc corrections

Holm-Bonferroni
Step-down method applied across all per-timepoint pairwise comparisons. Less conservative than Bonferroni, more powerful. Generally the recommended choice.
Bonferroni
Most conservative. Multiplies each p-value by the total number of comparisons.
Šidák
Similar to Bonferroni but slightly less conservative. Uses the formula 1−(1−p)m.

Normality & outlier checks

Anderson-Darling normality test
Tests whether data follows a normal distribution. If p < 0.05, the data is likely non-normal and you should consider non-parametric tests. Use the "Check Normality" button below the test selector.
IQR outlier detection
Identifies values beyond 1.5× the interquartile range from Q1 or Q3. These are the same bounds shown in box plots. Use the "Check Outliers" button to see per-group results.
Tip: If unsure about normality with small samples (n < 30), non-parametric tests are a safe choice. They are slightly less powerful but make fewer assumptions. Use the "Check Normality" button to run an Anderson-Darling test on your data.
